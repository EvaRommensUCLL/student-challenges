Programming - Car

Het doel van deze challenge is een eerste kennismaking met programmeren. Je leert een auto stapsgewijs rondrijden in een doolhof. De oefeningen bouwen op van moeilijkheid, we verwachten niet dat je alle oefeningen kan oplossen. Probeer zo ver mogelijk te geraken.

Open tests.html in een browser en lees de instructies om van start te gaan.

Indienen

Om deze challenge in te dienen maak je een screenshot in de browser van tests.html om te laten zien hoe ver je geraakt bent (de groene vakjes in de linkerkolom).
